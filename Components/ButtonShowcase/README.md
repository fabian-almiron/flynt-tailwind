# Button Showcase

A comprehensive showcase component that demonstrates all button types and variants available in the Canopy theme.

## Features

- **Solid Buttons**: Primary, Secondary, Success, Dark, Light variants
- **Ghost Buttons**: Outline buttons that fill on hover
- **Button Sizes**: Small, Default, Large, XL (for pricing pages)
- **Special Buttons**: Animated, Loading, Progressive (with arrows)
- **Progressive Forms**: Interactive email capture fields
- **Button Groups**: Grouped and attached button layouts
- **Icon Integration**: Buttons with icons from the 391-icon system
- **Theme Support**: Automatically adapts to light/dark themes

## Usage

This component is perfect for:
- Style guides and design system documentation
- Testing button implementations
- Demonstrating button capabilities to clients
- Development reference

## Fields

- **Title**: Optional title for the showcase section
- **Theme**: Light/Dark theme selection for the component

## Dependencies

- Icon system (391 icons)
- Button system styles
- Theme switcher functionality
