# Icon System Showcase

A comprehensive showcase component that demonstrates the icon system and theme switching capabilities of the Canopy theme.

## Features

- **Icon System Demo**: Display of all 391 available icons
- **Theme Switcher**: Interactive light/dark mode toggle
- **Brand Colors**: Showcase of the complete Canopy color palette
- **Icon Usage Examples**: Different icon sizes, colors, and applications
- **Interactive Elements**: Live theme switching demonstration

## Usage

This component is perfect for:
- Demonstrating the icon system capabilities
- Testing theme switching functionality
- Showcasing brand colors and design system
- Development reference and testing

## Fields

- **Title**: Optional title for the showcase section
- **Theme**: Light/Dark theme selection for the component

## Dependencies

- Icon sprite system (391 icons)
- Theme switcher JavaScript
- Brand color system
- Light/Dark theme styles
