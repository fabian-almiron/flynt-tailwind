# Block: Text & Image

A flexible content block that displays text content alongside an image.

## Features

- **Layout**: Swap image position (left â†” right)
- **Pre-title**: Optional small text above the main title
- **Title**: Main heading
- **Content**: Rich text content area
- **Bullet List**: Optional bullet list with checkmark icons
- **Button**: Optional call-to-action button
- **Color Options**: Customizable background and text colors
- **Theme Support**: Light/dark theme variations

## Fields

### Content Tab
- **Layout**: Choose image position (left or right)
- **Pre-title**: Small uppercase text (optional)
- **Title**: Main heading (required)
- **Content**: WYSIWYG editor for body text (required)
- **Bullet List**: Repeater field for list items (optional)
- **Button**: Link field for CTA button (optional)
- **Image**: Featured image (required)

### Options Tab
- **Background Color**: Custom background color
- **Text Color**: Custom text color
- **Theme**: Light/dark theme selection

## Usage

This component gracefully handles the absence of optional elements (pre-title, bullet list, button) without leaving empty space or placeholders.
